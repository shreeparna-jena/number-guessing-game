# Number Guessing Game

A simple Python game where the computer picks a random number and the user tries to guess it.

## How it works
- The program chooses a random number between 1 and 100
- The user guesses the number
- The program gives hints (higher or lower)
- The game ends when the correct number is guessed

## Requirements
- Python 3

## How to run
1. Run the script
2. Enter your guesses when prompted
